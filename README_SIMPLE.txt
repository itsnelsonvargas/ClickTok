Q: Why do I need to install dependencies?
A: ClickTok uses special libraries for video creation, TikTok automation,
   etc. These aren't part of standard Python.

Q: How long does installation take?
A: 5-10 minutes (downloading ~650 MB of packages)

Q: Do I need to install every time?
A: No! Only ONCE. After that, just double-click ClickTok.bat

Q: Is it safe?
A: Yes! All packages come from official Python Package Index (PyPI)

Q: What if installation fails?
A: See PYTHON_313_FIX.txt (likely Python version issue)

Q: Can I skip installation?
A: No, ClickTok won't work without dependencies
